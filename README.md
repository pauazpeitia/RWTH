# RWTH
This repository contains three key projects developed during my Computer Science B.Sc.:

1. **SPP: ML for Medical Technology** – A low-code platform for ML pipelines using Django, React, and Kubernetes.
2. **3D Shape Analysis: PointNet** – Investigation and analysis of Deep Learning on unstructured 3D data.
3. **Constraint Logic Programming** – Investigation of solver efficiency and CLPFD in Prolog.
